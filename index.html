About Me

Welcome! I'm Derrick, a passionate engineer and programmer with a love for problem-solving and creating meaningful solutions. I have a diverse skill set spanning Python, C++, VBA, and more, with experience in robotics, data science, and engineering applications. Below are some of the most notable projects I've worked on.

Projects

Ball Collection Robot (Python)

As part of the programming team, I developed a logic system for a robotic ball collection task. I designed a mechanism to detect and autocorrect ball detection overshoots using sensor timing and created a dynamic travel distance algorithm based on ultrasonic sensor readings.

Bermuda Triangle Data Science (Python)

Analyzed data containing hundreds of thousands of latitude and longitude points to investigate abnormalities in the Bermuda Triangle. I implemented a custom algorithm to filter points within the triangle for statistical analysis, uncovering insights about potential anomalies.

Intruder Detection Arm (Python)

Led the machine learning efforts using YOLOv3 to detect intruders. Developed a 2D-to-3D coordinate conversion system to enable inverse kinematics for precise robotic arm movements.

NFC Vending Machine (C++)

Designed and CNC-cut components for a custom NFC vending machine. Developed efficient logic for handling NFC scanning processes. The machine was designed to be modular and dynamic, avoiding hardcoding for enhanced operation flexibility. This project won the Gold Award among 25 teams.

VBA Calculator (VBA)

During my internship, I analyzed 20 years of raw operational data to create a dynamic Excel-based calculator. This tool tracks individual component lifespans and usage, enabling the company to make data-driven maintenance decisions. The calculator is still actively used today.
