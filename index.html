Projects

Ball Collection Robot (Python - Vex Robotics)

I developed a logic system for a robotic ball collection task. Logic was coded to detect and autocorrect ball detection overshoots using sensor timing and I also created a dynamic travel distance algorithm based on ultrasonic sensor readings.

Bermuda Triangle Debunking Using Data Science (Python - Jupyter)

Analyzed data containing hundreds of thousands of latitude and longitude points to investigate abnormalities in the Bermuda Triangle. I developed a novel algorithm to filter points within a triangular shape for statistical analysis to uncover insights about potential anomalies.

Intruder Detection Arm (Python)

Led the machine learning efforts using YOLOv3 to detect intruders. Developed a novel 2D-to-3D coordinate conversion system to enable inverse kinematics for precise robotic arm movements.

Contactless NFC Pharmacy (C++ - Arduino)

Designed and CNC-cut components for a custom NFC vending machine. Developed efficient, soft-coded logic for handling NFC scanning processes.This project won the Gold Award among 25 teams.

VBA Calculator (Office VBA)

I analyzed 20 years of raw operational data to create a dynamic Excel-based calculator and figure generator. This tool tracks individual component lifespans and usage to enable the company to make data-driven maintenance decisions.
